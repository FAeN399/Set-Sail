# Set-Sail Roadmap

## Milestone 1 — Bootstrap the Ship
- Finalize repo scaffolding, documentation, and automation.
- Publish the inaugural voyage log.

## Milestone 2 — First 3 Voyages
- Ship three small experiments across prompts, code, and workflows.
- Capture logbook entries and ensure artifacts are reusable.

## Milestone 3 — Reusable Voyage Template
- Expand `scripts/bootstrap.ts` with multiple templates and metadata prompts.
- Provide examples that demonstrate the voyage workflow end-to-end.

## Milestone 4 — Publishing the Logbook
- Automate publishing voyage summaries to a public site or README section.
- Invite contributors and gather feedback for future explorations.
